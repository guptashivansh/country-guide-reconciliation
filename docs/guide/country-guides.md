# Country Guides

The country guide pages are the client-facing output of the reconciliation pipeline.

## Guide List

![Guide List](../assets/screenshots/guide_list.png)

The guide list (`/guide`) shows a card grid of all monitored countries with:

- Country flag
- Total rule count
- Last-updated date

## Country Detail

![Country Guide Detail](../assets/screenshots/guide_country.png)

Each country page (`/guide/<country>`) displays the full employment guide organized into 7 categories:

| Category | Examples |
|----------|----------|
| Leave | Annual leave, sick leave, parental leave, public holidays |
| Working Hours | Standard hours, overtime rules, rest periods |
| Compensation | Minimum wage, pay frequency, bonus requirements |
| Benefits | Health insurance, pension, social security |
| Employment | Contracts, probation, termination, notice periods |
| Immigration | Work permits, visa types, sponsorship |
| Safety | Workplace safety, occupational health |

### Features

- **Sticky sidebar** — Quick navigation between sections
- **Last-updated timestamps** — Per-rule update tracking
- **Light/dark mode** — Toggle via the theme switcher
- **Source attribution** — Each rule links back to its official source
